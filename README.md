# soahomework_commerce
SOA课程第二次作业的实现部分 

### 运行环境:  
1. Apache Tomcat 7  
2. MySQL 5.x
  

### 数据库:
MySQL 5.x
地址: localhost
端口: 3306
数据库名: soaworktwo
用户名: soa
密码: soa  

### 初始化: 
数据库: 在mysql运行init_db.sql  

### 前后端交互:  
前端: JavaScript & Ajax & Json
后台: Jax-RI & Jax-WS & Json

### Notice:  
开发分支: developer
可以根据需要自建其他分支进行开发，之后合并到developer
不要直接push到master

服务接口非标准化描述见Wiki或项目中的《SOA\_ASS2\_E-Commerce\_服务说明(更新中).md》
